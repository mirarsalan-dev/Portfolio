# Premium Engineering Portfolio

A high-performance, cinematic developer portfolio built with **Python (Flask)** and **Tailwind CSS**. 

## 🚀 Key Features
- **Dynamic Background:** Interactive starfield animation engine.
- **Glassmorphism UI:** Modern, frosted-glass design with dark mode.
- **Automated Resume:** Dynamic PDF generation using `ReportLab`.
- **GitHub Integration:** Live repository fetching via GitHub API.
- **Scroll Reveal:** Smooth intersection-observer based animations.

## 🛠️ Tech Stack
- **Backend:** Flask (Python)
- **Frontend:** Tailwind CSS, JavaScript (ES6+)
- **Tools:** Chart.js, ReportLab, Lucide Icons

## ⚙️ Setup
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/your-repo-name.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python run.py`
